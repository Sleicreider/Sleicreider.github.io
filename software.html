Sleix is a small Software, which makes it possible to see incoming calls on your Windows PC. Practical for Gamers and also Standard Windows users. With its integrated Alarm Function you won't miss any incoming calls anymore.

Requirements:

-Android Smartphone
-Minimum Android Version 2.1
-Windows PC
-Active WiFi connection on botch devices
-Both device must be in the same WiFi network
