Hazelnut is a 2D Mobile Game (Android & iOS)
The player must try to catch all the dropping walnuts
which are dropped by the Evil Terrifying Squirrel.
The squirrel also drops evil objects, which shouldn't be catched!

Using:
  • C++
  • cocos2d-x Engine
  • custom cocos2d-x Framework devoped by Dominik Reisner
